TaskShare

Finally, an easy-to-share task list tool.

Dead-simple task/todo lists you can access and share with others online. Designed for primarily mobile browsers.

Create Taskboard Now

Simple Sharing

Want to share your lists with someone? Send them the page address. Done.

Responsive

Works well on almost all mobile devices as well as on your desktop.

Open-Source

TaskShare is released under an MIT license. Feel free to grab it from the Github repo and host an instance yourself.

Easy to Use

Other todo apps get bogged down in unnecessary features. TaskShare focuses on making todo lists that are easy to access, use and share. That's it.

Built on the Fat-Free Framework

TaskShare utilizes the Fat-Free PHP Framework, a lightweight (only 56kb!) framework.
